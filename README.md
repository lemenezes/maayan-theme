# Maayan Theme - Modern SaaS Premium

Um child theme minimalista e moderno para ListingHive, com design SaaS premium.

## 📋 Características

✨ **Paleta de Cores Premium**
- Azul Céu: `#0EA5E9`
- Roxo Médio: `#9333EA`
- Branco Limpo: `#F8FAFC`

✨ **Componentes Modernos**
- Cards com sombra sutil e hover elegante
- Grid sutil no background (100px × 100px)
- Botões arredondados com gradientes
- Inputs com focus elegante
- Tipografia moderna e limpa

✨ **Layout Responsivo**
- Totalmente mobile-friendly
- Grid layouts flexíveis
- Acessibilidade (WCAG 2.1)

## 🚀 Instalação

1. Baixe o arquivo `maayan-theme.zip`
2. Acesse **Aparência → Temas → Adicionar novo → Enviar tema**
3. Selecione o arquivo ZIP
4. Clique em **Ativar**

## 📁 Estrutura

```
maayan-theme/
├── style.css           # Header do tema
├── functions.php       # Enfileiração de CSS
└── assets/css/
    └── custom.css      # Estilos principais
```

## 🎨 Customizações

Para customizar as cores, edite as variáveis CSS em `custom.css`:

```css
:root {
  --color-primary: #0EA5E9;      /* Azul Céu */
  --color-secondary: #9333EA;    /* Roxo */
  --color-accent: #06B6D4;       /* Ciano */
  /* ... */
}
```
